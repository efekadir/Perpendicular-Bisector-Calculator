# Perpendicular-Bisector-Calculator

A Java console application that finds the equation of the perpendicular bisector between two points in a 2D coordinate system. 

## Example Output
```text
Enter x1 coordinate: 0
Enter y1 coordinate: 0
Enter x2 coordinate: 4
Enter y2 coordinate: 4
The equation of the perpendicular bisector is: y = -1.0x + 4.0
